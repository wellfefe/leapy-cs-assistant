# Criação de conta

## Cadastro

Para criar uma conta na plataforma Leapy, o usuário deve informar nome,
e-mail corporativo, empresa e senha.

## Confirmação do e-mail

Após o cadastro, o usuário receberá um link de confirmação por e-mail.
O link possui validade de 24 horas.

## Problemas com o cadastro

Caso o e-mail não seja recebido, o usuário deve verificar a pasta de spam
ou solicitar um novo envio.
