# Recuperação de senha

## Solicitação

Na tela de login, o usuário deve selecionar a opção "Esqueci minha senha"
e informar o e-mail cadastrado.

## Validade do link

O link de redefinição de senha possui validade de 30 minutos.

## Link expirado

Caso o link expire, o usuário deverá realizar uma nova solicitação.
